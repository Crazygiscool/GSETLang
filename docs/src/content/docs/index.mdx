## ⚡ What is GSET?

GSET (**Generic Syntax Extension Tool**) is a **universal transpiler** that bridges programming languages. Write code using your favorite syntax, then compile and run it on any runtime! 🎯

<CardGrid stagger>
  <Card title="🌍 Any Syntax, Any Target" icon="rocket">
    Write in **Python** syntax and compile to **Java**, or use **Go** patterns in **JavaScript** code. The possibilities are endless!
  </Card>
  <Card title="🔄 Language Mapping" icon="puzzle">
    Define custom **keyword mappings** between source and target languages. Create your own DSL! ✨
  </Card>
  <Card title="⚙️ External Configuration" icon="setting">
    All settings managed through `gset.conf` - **no hardcoded values**. Stay flexible! 🔧
  </Card>
  <Card title="🛠️ Multi-Compiler Support" icon="rocket">
    Supports **Python**, **Node.js**, **Java**, **Go**, **Ruby**, **PHP**, and more! 📦
  </Card>
</CardGrid>

## 🚀 Quick Example

Write your code in any syntax you prefer:

```python
# Write in Python-style syntax
main() {
    data = [1, 2, 3, 4, 5]
    for item in data {
        print("Item:", item)
    }
}
```

Compile it to your target language. GSET handles the translation automatically! 🎉
