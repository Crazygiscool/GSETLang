## What is GSET?

GSET (Generic Syntax Extension Tool) is a transpiler that lets you write code in any language syntax and compile it to any target language's runtime. Whether you prefer Python's elegance, JavaScript's flexibility, or Go's simplicity, GSET lets you express your logic in your preferred syntax.

## Quick Example

Write your code in any syntax:

```python
main() {
    print("Hello from GSET!")
}
```

Compile it to your target language. GSET handles the translation.
