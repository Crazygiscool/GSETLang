GSET provides multiple loop constructs for iterating over data and executing repeated code.

## For Loop

Iterate over a sequence:

```go
function main() {
    for i = 0; i < 5; i = i + 1 {
        print(i)
    }
}
```

**Output:**

```
0
1
2
3
4
```

## For-In Loop (Foreach)

Iterate over list elements:

```go
function main() {
    nums = [1, 2, 3, 4, 5]
    
    for i in nums {
        print(i)
    }
}
```

**Output:**

```
1
2
3
4
5
```

## For-In with Index

Get both index and value:

```go
function main() {
    fruits = ["apple", "banana", "cherry"]
    
    for idx, val in fruits {
        print(idx)
        print(val)
    }
}
```

**Output:**

```
0
apple
1
banana
2
cherry
```

## While Loop

Loop while condition is true:

```go
function main() {
    count = 5
    
    while count > 0 {
        print(count)
        count = count - 1
    }
    
    print("Done!")
}
```

**Output:**

```
5
4
3
2
1
Done!
```

## Loop Control

### Break

Exit loop early:

```go
function main() {
    nums = [1, 2, 3, 4, 5]
    
    for i in nums {
        if i == 3 {
            print("Found 3, stopping")
            break
        }
        print(i)
    }
}
```

**Output:**

```
1
2
Found 3, stopping
```

### Continue

Skip to next iteration:

```go
function main() {
    nums = [1, 2, 3, 4, 5]
    
    for i in nums {
        if i % 2 == 0 {
            continue  # skip even numbers
        }
        print(i)
    }
}
```

**Output:**

```
1
3
5
```

## List Comprehensions

GSET supports Python-style list comprehensions:

```go
function main() {
    nums = [1, 2, 3, 4, 5]
    
    # Square each number
    squared = [x * x for x in nums]
    print(squared)  # [1, 4, 9, 16, 25]
    
    # Filter even numbers
    evens = [x for x in nums if x % 2 == 0]
    print(evens)  # [2, 4]
    
    # Complex comprehension
    result = [x * 2 for x in nums if x > 2]
    print(result)  # [6, 8, 10]
}
```

### Nested List Comprehensions

```go
function main() {
    matrix = [[1, 2], [3, 4], [5, 6]]
    
    # Flatten
    flat = [x for row in matrix for x in row]
    print(flat)  # [1, 2, 3, 4, 5, 6]
    
    # 2D transform
    doubled = [[x * 2 for x in row] for row in matrix]
    print(doubled)  # [[2, 4], [6, 8], [10, 12]]
}
```

## Common Patterns

### Iterate Range

```go
function main() {
    # Using while with counter
    for i = 0; i < 10; i = i + 1 {
        print(i)
    }
}
```

### Iterate Dictionary (via list)

```go
function main() {
    # Note: GSET uses lists, not dicts
    # Simulate key-value with list of pairs
    items = [["name", "Alice"], ["age", 30]]
    
    for pair in items {
        print(pair[0])  # key
        print(pair[1])  # value
    }
}
```

<Callout type="info">
List comprehensions are often more readable than explicit loops for simple transformations and filters.
</Callout>

## Performance Tips

1. **Use comprehension for simple cases**: More concise and often faster
2. **Prefer for-in for iteration**: Avoid manual index management
3. **Break/Continue judiciously**: Can make logic harder to follow
4. **Avoid nested loops**: Consider flattening or using comprehensions
