Build GSET from source to get the latest features or contribute to development.

## Prerequisites

- **Go 1.21 or later**
- **Git**
- **Make** (optional, but recommended)

Check your Go version:

```bash
go version
```

## Quick Build

```bash
# Clone the repository
git clone https://github.com/Crazygiscool/GSETLang
cd GSETLang

# Build the binary
go build -o gset .

# Verify
./gset --version
```

## Using Make

The project includes a Makefile for common tasks:

```bash
# Build
make build

# Install to $GOPATH/bin
make install

# Run tests
make test

# Run with race detector
make test-race

# Cross-compile for all platforms
make crossbuild

# Clean build artifacts
make clean
```

## Build Options

### Debug Build

Build with debug information:

```bash
go build -o gset-debug -tags debug .
```

### Release Build

Optimized release build:

```bash
go build -o gset -ldflags="-s -w" .
```

### Cross-Compilation

Build for different platforms:

```bash
# Linux
GOOS=linux GOARCH=amd64 go build -o gset-linux-amd64 .

# macOS (Intel)
GOOS=darwin GOARCH=amd64 go build -o gset-darwin-amd64 .

# macOS (Apple Silicon)
GOOS=darwin GOARCH=arm64 go build -o gset-darwin-arm64 .

# Windows
GOOS=windows GOARCH=amd64 go build -o gset-windows-amd64.exe .
```

## Development Setup

### 1. Clone and Setup

```bash
git clone https://github.com/Crazygiscool/GSETLang
cd GSETLang
```

### 2. Install Dependencies

```bash
go mod download
```

### 3. Run Tests

```bash
go test ./...
```

### 4. Build and Test

```bash
go build -o gset .
./gset run test/hello.gset
```

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `GSET_DEBUG` | Enable debug logging | not set |
| `GSET_CONFIG` | Custom config file path | `gset.conf` |
| `GSET_HOME` | GSET home directory | `~/.gset` |

### Debug Mode Example

```bash
GSET_DEBUG=1 gset run hello.gset
```

Output includes:
- Lexer tokens
- Parser decisions
- Transpiler output
- Detailed error messages

## Building for Distribution

### Creating Release Builds

```bash
# Create release directory
mkdir -p release

# Build all platforms
make crossbuild

# Or manually:
GOOS=linux GOARCH=amd64 go build -o release/gset-linux-amd64 .
GOOS=darwin GOARCH=amd64 go build -o release/gset-darwin-amd64 .
GOOS=darwin GOARCH=arm64 go build -o release/gset-darwin-arm64 .
GOOS=windows GOARCH=amd64 go build -o release/gset-windows-amd64.exe .
```

### Creating Archives

```bash
# Linux
cd release
tar -czf gset-linux-amd64.tar.gz gset-linux-amd64

# macOS
tar -czf gset-darwin-amd64.tar.gz gset-darwin-amd64

# Windows (PowerShell)
Compress-Archive gset-windows-amd64.exe gset-windows-amd64.zip
```

## Troubleshooting

### "go: command not found"

Install Go from [go.dev](https://go.dev/dl/) or your package manager.

### Permission denied on Linux/macOS

```bash
chmod +x gset
# or
sudo mv gset /usr/local/bin/
```

### Build fails with dependencies

```bash
go mod tidy
go mod download
```
