Welcome to the GSET documentation! This section will help you get up and running quickly.

## What is GSET?

GSET (Generic Syntax Extension Tool) is a transpiler that lets you write code in any language's syntax and compile it to run on any target language's runtime. Write once, run anywhere — with the syntax you prefer.

### Write in Familiar Syntax

```go

    print(i)
```

Choose your target: Python, JavaScript, Java, Go, Ruby, PHP, C#, Rust, Swift, Kotlin, C++, and more.

## Key Features

| Feature | Description |
|---------|-------------|
| **Multi-Syntax Support** | Write in Python, JavaScript, Java, Go, or custom syntax |
| **Multi-Target Output** | Transpile to 14+ programming languages |
| **Custom Keywords** | Define your own function names via headers or config files |
| **Security Hardened** | Input validation, command sanitization, depth limits |
| **List Comprehensions** | Expressive `[expr for x in list if condition]` syntax |
| **Pattern Matching** | Clean `match` statements for conditional logic |
