GSET (Generic Syntax Extension Tool) is a powerful transpiler that lets you write code using **any language's syntax** and compile it to run on **any target language's runtime**.

## Quick Example

Write this in any syntax you prefer:

```go
print("Hello from Python syntax!")
nums = [1, 2, 3]
for i in nums {
    print(i)
}
```

Compile to your target language:

| Target | Output |
|--------|--------|
| Python | `print("Hello from Python syntax!")` |
| JavaScript | `console.log("Hello from Python syntax!")` |
| Java | `System.out.println("Hello from Python syntax!")` |
| Go | `fmt.Println("Hello from Python syntax!")` |

## Features

- **14+ Language Targets**: Python, JavaScript, Java, Go, Ruby, PHP, C#, Rust, Swift, Kotlin, C++, and more
- **Familiar Syntax**: Write in your preferred programming style
- **Custom Keywords**: Define your own function names
- **List Comprehensions**: Expressive and concise array operations
- **Pattern Matching**: Clean match statements for conditional logic
- **Security Hardened**: Input validation, command sanitization, depth limits
