GSET supports transpiling to **14+ programming languages**. The target language is automatically detected from your file extension.

## Language Overview

| Extension | Target | Print Output | Array Syntax |
|-----------|--------|--------------|--------------|
| `.gset` | Go | `fmt.Println()` | `[]int{1, 2, 3}` |

## Auto-Detection

GSET automatically detects the target language from the file extension:

```bash
# Python target (default)
gset transpile hello.py

# JavaScript target
gset transpile hello.js

# Java target
gset transpile hello.java
```

## Manual Selection

You can also specify the target explicitly:

```bash
gset transpile --target python hello.gset
gset transpile --target javascript hello.gset
```

### Go

```go
    fmt.Println("Hello, World!")
    nums := []int{1, 2, 3}
    for _, i := range nums {
        fmt.Println(i)
    }
}

func main() {
    fmt.Println("Hello, World!")
    nums := []int{1, 2, 3}
    for _, i := range nums {
        fmt.Println(i)
    }
}
```

### Python

```python
def main():
    print("Hello, World!")
    nums = [1, 2, 3]
    for i in nums:
        print(i)


main()
```

### JavaScript

```javascript
function main() {
    console.log("Hello, World!");
    const nums = [1, 2, 3];
    for (const i of nums) {
        console.log(i);
    }
}

main();
```

### Java

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        int[] nums = new int[]{1, 2, 3};
        for (int i : nums) {
            System.out.println(i);
        }
    }
    public static void main() {
        System.out.println("Hello, World!");
        int[] nums = new int[]{1, 2, 3};
        for (int i : nums) {
            System.out.println(i);
        }
    }
}
```

## Choosing a Target

Consider these factors when choosing a target language:

| Factor | Recommendation |
|--------|---------------|
| Web development | JavaScript/TypeScript |
| Systems programming | Go, Rust, C++ |
| Enterprise software | Java, C# |
| Scripting/automation | Python, Ruby |
| Mobile apps | Swift, Kotlin |
